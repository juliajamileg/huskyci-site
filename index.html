Security Tests Available

Add a security stage into your pipeline

Let HuskyCI client perform the required requests to the API by simply downloading and running it.

Get detailed information from issues

If a vulnerability is found in a new push, HuskyCI shows the severity, the file, the line, and many other useful information that could help developers fix it.

Fail the pipeline if vulnerabilities are found

Avoid deploying vulnerable code! HuskyCI's stage will fail in case a new vulnerability is pushed into the repository before reaching production or even development environment.
